// Installation Instructions
Step 01

Import the repository's GPG public key to authorize package verification:

sudo wget -O /usr/share/keyrings/es-de-archive-keyring.gpg https://OWNER.github.io/REPO/public.gpg
Step 02

Add the APT source list entry for Debian Trixie:

echo "deb [signed-by=/usr/share/keyrings/es-de-archive-keyring.gpg] https://OWNER.github.io/REPO/ trixie main" | sudo tee /etc/apt/sources.list.d/es-de.list
Step 03

Update your package lists and install ES-DE:

sudo apt update && sudo apt install emulationstation-de
i
How this works: Packages in this repository are compiled directly from the official ES-DE GitLab source tags inside a native Debian Trixie environment using GitHub Actions. The repository indices are automatically generated, signed using GPG, and updated weekly.

This is an unofficial repository. Visit the official ES-DE project site at es-de.org.

Maintained automatically via GitHub Actions.
